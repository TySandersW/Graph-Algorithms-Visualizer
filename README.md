# Graph-Algorithms-Visualizer
A project that shows essential CS algorithms visualized in C++ code, using algoriths such as BFS, DFS, Dijkstra's and Prim and Kruskal MST and A* argorithms.

While it is under construction right now, the code will visualize traversing order and shortest past with terminal based animations to demonstrate algorithm logic, and will use priority queues for best case algorithm effeciency.

This code also uses SFML, designed to open windows to run the program visually. I've had to become familiar with programming with SFML, using its basic concepts and incoorperating into my code, because as someone who is eager to learn new things, using SFML is exciting to me in order to improve my code.
